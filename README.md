# ECOMMERCE-EXCEL-DASHBOARD
Interactive Excel E-Commerce Sales Dashboard built using Pivot Tables, Pivot Charts, Sparklines, and slicers. Tracks key KPIs: Sales, Profit, Quantity, Orders, and Profit Margin with YOY growth. Includes monthly sales analysis, category-wise profit, sales share, state-wise map, and top sub-categories. Fully editable and ready for custom data.
Easily customizable **Excel E-Commerce Sales Analysis Dashboard** built end-to-end in Excel (no Power BI required).
This dashboard tracks overall performance and year-over-year growth for an online store, and can be plugged into any similar sales dataset.

### Overview

The file contains a fully interactive dashboard to monitor:

* **Total Sales**
* **Total Profit**
* **Quantity Sold**
* **Number of Orders**
* **Profit Margin**
* **YOY Growth % for all key KPIs**

### Main Dashboard Views

* **Sales & Profit Analysis (by Month)**
  Dual-axis column chart showing monthly *Sales* vs *Profit* trends.

* **Category-Wise Profit (Waterfall Chart)**
  Visual breakdown of profit contribution from **Technology, Office Supplies, Furniture**, and overall grand total.

* **Category-Wise Sales Share % (Donut Chart)**
  Segment share of total sales by category with central KPI card.

* **Sales by State (Map Chart)**
  Geo heat map that highlights sales performance across regions/states.

* **Top 5 Sub-Categories by Sales**
  Horizontal bar chart ranking the best-performing product sub-categories (e.g., Chairs, Phones, Binders, Storage, Tables).

* **Filter Panel (Interactive Slicers)**
  Dynamic filters for:

  * **Year** (2011â€“2014 or your own range)
  * **Region** (Central, East, South, West)
  * **Segment** (Consumer, Corporate, Home Office)

### Tech & Features

* Built entirely in **Microsoft Excel** using:

  * Pivot Tables & Pivot Charts
  * Slicers for interactive filtering
  * Sparklines for KPI trend lines
  * Formulas & named ranges for calculations
* Fully editable layout and formatting so you can:

  * Replace the sample data with your own dataset
  * Add/remove KPIs or visuals
  * Re-brand the theme for your organization

### How to Use

1. Open the workbook in Excel (desktop version recommended).
2. Replace the sample data in the **Data** sheet with your own e-commerce/exported sales data.
3. Refresh all Pivot Tables/Charts (`Data > Refresh All`).
4. Use the slicers (Year, Region, Segment) to interactively explore performance.

You can fork this repo to adapt the dashboard for your own reporting needs or extend it with additional KPIs and visuals.
