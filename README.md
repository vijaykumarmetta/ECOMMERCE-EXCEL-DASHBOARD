# ECOMMERCE-EXCEL-DASHBOARD
Interactive Excel E-Commerce Sales Dashboard built using Pivot Tables, Pivot Charts, Sparklines, and slicers. Tracks key KPIs: Sales, Profit, Quantity, Orders, and Profit Margin with YOY growth. Includes monthly sales analysis, category-wise profit, sales share, state-wise map, and top sub-categories. Fully editable and ready for custom data.
